# Onchain Loyalty Program

It is a decentralized loyalty rewards application where businesses create custom loyalty programs with themed stamp designs and customers collect stamps to redeem tokens.

## Features

For Businesses:
1. Custom Loyalty Programs
2. Flexible stamps and token reward amounts
3. Dashboard with customers loyalty tracking

For Customers
1. Multi-card wallet
2. Token Redemption

## Start with:
- Node.js v16
- Phantom wallet extension

## As a business owner
1. Sign up with business name and type of business
2. Configure the program as setting stamps required and token reward
3. Share your email with customers to start loyalty program
4. Add stamps with the customer code provided by customer

## As a customer
1. Sign up
2. Add loyalty cards with the business owner's email
3. Share your customer code with business owner to collect stamps
4. Redeem token rewards after the card loyalty card is full
